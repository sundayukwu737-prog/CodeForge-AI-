# CodeForge AI 🚀

An AI-powered coding education platform built for the DEV Weekend Challenge: Passion Edition.

## The Passion Behind It
Programming is more than just a skill to me—it's a passion and the career I want to build. I created CodeForge AI to solve the challenges I faced while learning to code by providing a place where developers can stay motivated, organize their learning, and transform ideas into real projects. My hope is that it inspires others to keep learning and building, just as it inspires me every day.

## Features
- 📊 Developer Dashboard with XP, streaks, and achievements
- 🗺️ Interactive Learning Roadmap
- 💡 AI Project Generator (powered by Google Gemini)
- 🤖 AI Assistant chat (powered by Groq / Llama 3.3)
- 📚 Developer Resources library

## Tech Stack
- HTML, CSS, JavaScript (vanilla, no frameworks)
- Google Gemini API (gemini-2.5-flash) — Project Generator
- Groq API (llama-3.3-70b-versatile) — AI Assistant chat
